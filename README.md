# Liquid Time-Constant (LTC) Finance Bots

Этот проект представляет собой набор автономных торговых скриптов-оракулов для криптобиржи Binance, построенных на базе новейшей архитектуры «жидких» нейросетей (Liquid Time-Constant, LTC).

## Особенности проекта
- **Тру-Реалтайм:** Скрипты получают данные с биржи Binance через WebSocket с задержкой в миллисекунды.
- **Обучение на лету (Online Learning):** Нейросеть не нужно обучать часами на исторических данных. Она постоянно корректирует свои веса (`optimizer.step()`) прямо во время работы, приспосабливаясь к текущему тренду.
- **Сохранение памяти:** Все скрипты умеют сохранять свой "мозг" (веса модели) на диск и загружать его при перезапуске.
- **Интерактивные графики:** Визуализация реальной цены, прогноза и торговых сигналов в реальном времени с помощью `matplotlib`.
- **Мульти-фичи:** Нейросеть принимает на вход 2D тензор `[цена, объём]`, что позволяет учитывать силу тренда при прогнозировании.
- **Streamlit UI:** Веб-интерфейс с автообновлением для мониторинга торговли в реальном времени.

## Файлы в проекте

| Файл | Описание |
|------|----------|
| `streamlit_app.py` | Веб-интерфейс (Streamlit) с торговым ботом. Использует WebSocket `@trade`, собирает свечи вручную, обучает LTC на лету. Фичи: цена + объём. |
| `finance_bot_realtime_graph.py` | Базовый анализатор тиков (сделок). Анализирует каждую сделку на бирже, рисует сверхточный график с предсказанием на 1 тик вперед. Идеален для понимания работы LTC в реальном времени. |
| `finance_bot_trader.py` | Высокочастотный торговый бот (HFT). Работает по 1-секундному таймфрейму. Имеет встроенную логику учета комиссий и подает сигналы `BUY` (в лонг) / `SELL` (в шорт) прямо на графике. |
| `finance_bot_trader_minutes.py` | Торговый бот для более крупных таймфреймов (1 минута, 5 минут и т.д.). Умеет моментально подгружать исторические данные через REST API Binance, чтобы не ждать накопления свечей. |

## Установка
```bash
pip install torch ncps scikit-learn matplotlib websocket-client requests pandas pandas_ta streamlit
```

## Запуск Streamlit UI
```bash
streamlit run streamlit_app.py
```
Откроется браузер с веб-интерфейсом бота.

## Конфигурация
- `TIME_WINDOW_MINUTES` — таймфрейм свечей (по умолчанию 1 мин)
- `NEURONS` — количество нейронов в LTC (по умолчанию 32)
- `SEQ_LENGTH` — длина окна памяти (по умолчанию 50)
- `NUM_FEATURES` — количество фичей на входе (2: цена + объём)
