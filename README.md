# Liquid Time-Constant (LTC) Finance Bots

Этот проект представляет собой набор автономных торговых скриптов-оракулов для криптобиржи Binance, построенных на базе новейшей архитектуры «жидких» нейросетей (Liquid Time-Constant, LTC).

## Особенности проекта
- **Тру-Реалтайм:** Скрипты получают данные с биржи Binance через WebSocket с задержкой в миллисекунды.
- **Обучение на лету (Online Learning):** Нейросеть не нужно обучать часами на исторических данных. Она постоянно корректирует свои веса (`optimizer.step()`) прямо во время работы, приспосабливаясь к текущему тренду.
- **Сохранение памяти:** Все скрипты умеют сохранять свой "мозг" (веса модели) на диск и загружать его при перезапуске.
- **Интерактивные графики:** Визуализация реальной цены, прогноза и торговых сигналов в реальном времени с помощью `matplotlib`.

## Файлы в проекте
В проекте оставлены только самые продвинутые версии скриптов:

1. `finance_bot_realtime_graph.py` — Базовый анализатор тиков (сделок). Анализирует каждую сделку на бирже, рисует сверхточный график с предсказанием на 1 тик вперед. Идеален для понимания работы LTC в реальном времени.
2. `finance_bot_trader.py` — Высокочастотный торговый бот (HFT). Работает по 1-секундному таймфрейму. Имеет встроенную логику учета комиссий и подает сигналы `BUY` (в лонг) / `SELL` (в шорт) прямо на графике.
3. `finance_bot_trader_minutes.py` — Торговый бот для более крупных таймфреймов (1 минута, 5 минут и т.д.). Умеет моментально подгружать исторические данные через REST API Binance, чтобы не ждать накопления свечей.

## Установка
```bash
pip install torch ncps scikit-learn matplotlib websocket-client requests
```

## Запуск
Для запуска скрипта просто выполните:
```bash
python3 finance_bot_trader_minutes.py
```
*(Графическое окно откроется автоматически)*

## Предупреждение о рисках
Данный проект является исследовательской (Research) работой в области применения LTC нейросетей в финансах. Код не содержит прямого подключения к торговому API Binance (отправки реальных ордеров). Любое использование этих прогнозов для торговли на реальные деньги осуществляется на ваш страх и риск.
